Паттерны ООП
============
В данном разделе выкладываются, как памятка, примеры и описания основных паттернов.
Паттерн - типичный образец шаблона решения/архитектуры часто встречающейся задачи.

На текущий моменты выложены следующие паттерны:
* Abstract Factory
* Adapter
* Bridge
* Builder
* Chain of Responsibility
* Command
* Composite 
* Decorator
* Factory Method
* Facade
* Iterator
* Object Pool
* Observer
* Prototype
* Proxy
* Simple Factory
* Singleton
* State
* Strategy
* Template Method
* Visitor

Еще планируются:
* Registry
* Flyweight
* Interpreter
* Mediator
* Memento
